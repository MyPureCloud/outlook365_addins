Install Instructions

  1. Log into OWA
  2. Change the url from /owa to /ecp
  3. Select apps from the options list
  4. Find the drop down and select Add from Url


    If you can't see that setting, you need to request from IT to enable it. You can refer them to case INC0099689 where I had to ask for access.
  5. Enter this url of the app to install:
    Directory http://purecloudoffice365.herokuapp.com/directory.xml
    Content Management http://purecloudoffice365.herokuapp.com/contentManagement.xml
    Voicemail http://purecloudoffice365.herokuapp.com/voicemail.xml
  6. Enjoy
